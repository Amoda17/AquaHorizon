Splash Screen

Technical Description

1.FullScreen setup -
body and html tags : These define the full structure of the page.
meta name="viewport" : Ensures responsive behavior on different screen sizes.

    Key CSS Properties -:
  • height: 100%; width: 100%; - Makes sure the entire viewport is covered.
  • display: flex; justify-content: center; align-items: center; - Centers all elements in the viewport.
  • overflow: hidden; - Prevents scrolling to maintain a clean fullscreen effect.

2.Background Video -
The source attribute in html is used here to define the video file and format.

    Key CSS Properties -:
  • position: fixed; top: 0; left: 0; - Ensures that the video in place across all screen sizes.
  • width: 100%; height: 100%; - Ensures full coverage of the screen.
  • z-index: -2; - Places the video behind all other content.

3.Dark Overlay -
The dark overlay is used for readability. A Class is defined as "overlay" by class attribute using the div element in html. A semi transparent layer over the vidoe is created for better readability.

    Key CSS Properties -:
  • background: rgba(0, 0, 0, 0.5) - A dark overlay is created with 50% opacity for better text readability.
  • z-index: -1 - Overlay is placed above the video but behind the text.

4.Wave Loader for Visual Feedback -
Class attribute in html is used here to create the class "wave-loader", which Contains the wave animation and "Loading..." text. Each animated wave is represented by using the class "wave" using a span element.

5.Animated text -
The phrase "Keeping our oceans alive for future generations!" is animated here for more visual engagement. A class called "animated-text" is defined to contain the animated text.

    Key CSS Properties -:
  • fadeSlide Animation makes the text fade in and slide up.
  • float Animation is used to create a bouncing effect.

    • 6.Mission Statement Box -
    Mission staetement is highlighted within a semi-transparent box by forming a class attribute named "mission-box".

      Key CSS Properties -:
    • background: rgba(255, 255, 255, 0.8); - Creates a semi-transparent box.
    • border-radius: 10px;- Gives rounded corners for a modern look.

      • 7.Automatic Redirect to Home Page -
      For automatic redirection java script concepts are used .

        Key java script Properties -:
      • setTimeout(function(){window.location.href = "home.html"; - This will be redirected to home page after 4 seconds. This ensures seamless transition without manual input.

        Accessibility

        The title and mission statement use high-contrast colors (white text on a dark overlay) to ensure readability. A glow animation is used for the title to enhance visibility without excessive motion The dark overlay on the video background ensures that white text remains readable. The mission box and group members box have a semi-transparent background with good contrast against text. The splash screen automatically redirects, but users with slower assistive technologies are not blocked from accessing the homepage.

Volunteer Page

Technical Description

The Volunteer Page of AquaHorizon is designed with a clear structure, interactive features, and user-friendly navigation. It effectively encourages participation in ocean conservation efforts through well-organized sections, visually appealing elements, and responsive design.

1.Page Structure and Layout
The Volunteer Page is structured to provide an engaging experience, allowing users to explore different volunteer opportunities.

    Key Html tags -:
  • <"header"> tag - Contains the website’s navigation and branding for consistency.
  • <"main"> tag - Wraps all the main content to separate it from the navigation and footer.
  • <"section"> tag - Organizes different parts of the page, such as the introduction, projects, and reviews.
  • <"h1"> to <"h3"> tags - Defines a clear heading hierarchy, improving readability.
  • <"button"> tag - Allows users to join volunteer activities with a clear call-to-action.

      • Key CSS Properties -:
    • background: rgba(255, 255, 255, 0.8);- applies a semi-transparent overlay for a clean design.
    • display: flex; and display: grid;- for responsive alignment of content.
    • padding: 20px; - provides spacing for better readability and organization.

      • 2.Volunteer Introduction Section
      A welcoming and informative first impression is created by this volunteer introduaction section.

        Key Html tags -:
      • <"div">element: Wraps text elements for styling and alignment.
      • <"h1"> tag : Displays the main title “Volunteer with AquaHorizon”.

          • Key CSS Properties -:
        • margin-top: 200px; - ensures the content appears below the navigation bar.
        • font-size: 45px; font-weight: bold; - highlights the call to action.

          • 3.Volunteer Programs and Scrollable Section
          The scrollable section is used to allow users to explore multiple volunteer activities without cluttering the page.

            Key Html tags -:
          • <"section"> tag - Groups related volunteer projects together.
          • <"div class=card"> - Wraps each volunteer opportunity, providing clear separation.
          • <"button"> tag A button is there in each volunteer programme created with hover effects which allows users to join a program or donate.

              • Key CSS Properties -:
            • display: grid; with grid-template-columns: repeat(2, 1fr); - organizes the projects into a responsive grid layout.
            • max-height: 500px; overflow-y: auto; - enables smooth vertical scrolling.

              • 4. Sorting Feature for Volunteer Projects
              A dropdown menu is used here to allow users to sort volunteer opportunities by popularity, recent activities, or impact.

                Key Html tags -:
              • <"select"> tag - Allows users to choose sorting options.
              • <"label"> tag - Provides a description for the sorting dropdown.
              • <"option"> tag - Lists different sorting choices like “Most Popular”, “Most Recent”, "high impact".

                  • Key CSS Properties -:
                • border: 1px solid #ccc; border-radius: 5px; - keeps the dropdown visually clean.

                  • 5. Sorting Feature for Volunteer Projects
                  The reviews section allows users to share their experiences, making the page more interactive and credible.Emojis are used to provide feeback easily.

                    Key Html tags -:
                  • <"div class=reviews-section"> - Wraps all reviews inside a dedicated container.
                  • <"textarea"> tag - Provides an input box for submitting feedback.
                  • <"button"> tag - Allows users to send their reviews.

                      • Key CSS Properties -:
                    • background: #f9f9f9; padding: 15px; border-radius: 10px; - creates clean and modern review cards.
                    • max-width: 300px; - ensures consistent card sizing.

                      • 6. Social Media Campaigns and Donations Section
                      The part includes social media initiatives and donation options to increase community engagement.

                        Key Html tags -:
                      • <"ul"> tag - Used to create an unordered list to choose different participation options for users.
                      • <"input type=number"> - Allows users to enter a number as the donation amount.

                          • Key CSS Properties -:
                        • transition: transform 0.3s ease-in-out; - Adds smooth hover effects.
                        • hover: transform: scale(1.05); - subtly increases button size on hover for interactivity.

                          • Accessibility

                          Alt Text for Images (alt="")is used if an image does not load, the alt text provides context. Descriptive alt text ensures screen readers can communicate the image's purpose. Dark text on a on light background improves readability. Labels help screen readers identify form elements. Without labels, users may not understand the purpose of a dropdown or input fields. Uses CSS media queries to adjust layout for smaller screens which makes the webpage responsive and mobile-friendly.

Content Page

Technical Description

1.Page Structure and Layout
The content page follows a logical HTML structure to ensure readability, accessibility, and smooth navigation.

    Key Html tags -:
  • <"main"> tag - Wraps the primary content, separating it from navigation and footer sections.
  • <"body">tag - Contains all the visible content of the webpage. Inside the body, sections are divided logically with <'div'>, <'section'>, <'h1'>, <'h2'>, and <'p'> tags.
  • <"section"> tag - Used for different thematic content blocks like Introduction, Causes, Effects, and Solutions. These sections provide clear separation for different topics.
  • <"img"> tag - Displays images with alt attributes for accessibility.
  • <"h1"> to <"h6"> tags - Defines a clear heading hierarchy, ensuring structured content.

      • Key CSS Properties -:
    • padding and margin - provide spacing for better readability.
    • display: flex; - ensures a responsive layout for structuring the content.

      2.Internal Navigation and "Go to Top" Button
      An internal navigation system improves user experience by allowing quick access to different sections of the page.

        Key Html tags -:
      • <"nav"> tag - Defines the internal navigation menu for seamless page movement.
      • <"a href=#section-id"> - Creates internal links to specific sections for easy navigation.
      • <"button"> tag - Used for the "Go to Top" button to enhance user convenience.

          • Key CSS Properties -:
        • position: fixed; - keeps the "Go to Top" button always visible.
        • transition: all 0.3s ease; - provides a smooth hover effect.
          • color: #0077cc; - Ensures links are distinct but readable.

          3.Internal Links for Seamless Navigation
          Internal links improve page usability, allowing users to navigate between related sections without reloading.

            Key Html tags -:
          • <"a href=#related-section"> - Directs users to another part of the same page.

              • Key CSS Properties -:
            • hover: text-decoration; - Enhances clarity on hover.

              • 4.Image Styling for Visual Appeal
              Images are integrated to enhance content engagement and illustrate marine pollution issues effectively.

                Key Html tags -:
              • <"img src=image.jpg alt=Description"> - Displays images with descriptions for accessibility.

                  • Key CSS Properties -:
                • width: 100%; max-width: 600px; - Ensures images scale responsively without breaking layout.
                • border-radius: 10px; - Applies rounded corners for a modern look.

                  • Accessibility

                  The color contrast between the text and background ensures readability. All paragraphs use justified alignment to maintain a clean reading flow. All images include alt attributes, providing meaningful descriptions for screen reader users. The internal navigation menu allows users to jump between sections. The "Go to Top" button is always visible, allowing easy navigation for users. The page uses CSS media queries to adjust content layout on smaller screens which makes the webpage more responsive and mobile-friendly.

Challenges and Lessons Learned

When adding the background video for the splashscreen ,the video was not covering the full screen properly. As a solution object-fit: cover; was used in CSS to ensure the video fills the entire viewport while maintaining its aspect ratio.I learned that Responsive design applies to videos as well—using object-fit: cover; prevents unwanted cropping or empty spaces.

Faced an issue with reading the text on the splash screen due to the dynamic background video. As a solution a semi-transparent overlay was added over the video using CSS.I learned that adding an overlay enhances text contrast and improves accessibility.What I have learnt here was that Small UI elements like dropdowns should follow the same color and theme as the website for a cohesive look.

Faced an issue with the sorting dropdown, where the sorting dropdown did not match the rest of the page design. As a solution CSS styling was customized to ensure consistency.

The internal navigation menu was hidden under the header at first when I run the code. As a solution the top property in the CSS was adjusted.What I learnt here was to make sure to leave enough space for the header or navigation bars when using fixed positioning.

The go-to-top button in the content page was not working properly at first. As a solution an anchor link (#top) was used instead of JavaScript.
